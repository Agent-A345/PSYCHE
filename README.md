## 🧠 PSYCHE — Personality Discovery Platform

An interactive personality assessment web application built using HTML, CSS, and JavaScript that helps users discover their personality type through immersive quizzes across career, character, and relationship categories. The app generates detailed personality reports, fictional character matches, radar charts, and shareable results.

## 🚀 Features
* **3 Quiz Modes** – Choose between Career, Character, and Relationship personality tests.
* **12-Question Personality Journey** – Complete a structured personality quiz experience.
* **Adaptive Question Logic** – Certain questions change dynamically based on previous responses.
* **Real-Time Timer** – Tracks time during quiz sessions.
* **Progress Tracking** – Shows question count and completion progress bar.
* **Answer Review System** – Review and edit answers before submission.
* **Radar Chart Analysis** – Visualizes personality trait distribution.
* **Detailed Personality Reports** – Displays type, tagline, description, strengths, and weaknesses.
* **Career Recommendations** – Suggests ideal career paths.
* **Hobby Recommendations** – Suggests suitable hobbies and passions.
* **Fictional Character Matching** – Matches users with fictional characters.

## 🎮 How to Use
**Choose Quiz Mode**
* Select Career, Character, or Relationship mode
* Pick the category you want to explore

**Start the Quiz**
* Click Begin the Test
* Quiz session starts immediately

**Answer Questions**
* Select answers for all personality questions
* Complete questions before timer runs out
* Track progress using progress bar

**Navigate Questions**
* Use Next button to move forward
* Use Previous button to revisit earlier questions

**Review Responses**
* Open review screen before submission
* Check unanswered questions
* Modify responses if needed
